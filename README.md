# 🚀 FounderAI

>AI_Powered Business Planning Platform for Enterpreneurs

## 🌐 Live Demo
👉 https://founder-ai-lake.vercel.app

## 📖 About

FounderAI is a full-stack AI-powered platform designed to help entrepreneurs, 
startups, and small business owners plan, analyze, and grow their businesses 
using Artificial Intelligence.

The platform leverages Google's Gemini AI to generate detailed business reports,
strategic insights, and marketing content in multiple languages,
enabling users to make smarter business decisions quickly.

## ✨ Features
- 🔐 Secure User Authentication
- 📊 AI Business Plan Generator
- 📈 AI SWOT Analysis
- 🎤 AI Pitch Deck Generator
- 📣 AI Marketing Campaign Generator
- 🏆 AI Competitor Analysis
- 🤖 AI Business Mentor Chat
- 📄 PDF Export
- 🌍 Multi-language Support

## 🛠️ Tech Stack
- React + Vite
- Node.js
- Express.js
- MongoDB Atlas
- Gemini AI API
- Tailwind CSS
- Render (Backend)
- Vercel (Frontend)
